Clothes4Happiness

Sistema de Gerenciamento de Produtos

 Visão Geral

O Clothes4Happiness é um sistema desenvolvido para otimizar o controle de produtos da loja de roupas homônima, oferecendo uma plataforma simples, eficiente e segura para o cadastro, edição, listagem e exclusão de itens.

O objetivo do sistema é apoiar a equipe de gestão da loja, reduzindo erros operacionais e facilitando o acompanhamento do estoque em tempo real.

 Objetivos Principais

Simplificar o cadastro e gerenciamento de produtos.

Permitir o controle de estoque e preços com mais agilidade.

Oferecer uma interface intuitiva e de fácil acesso para colaboradores.

Servir como base para futuras integrações com módulos de vendas e relatórios.

Tecnologias Utilizadas

Java 21

Spring Boot (Framework principal)

Spring Data JPA (acesso ao banco de dados)

Thymeleaf (motor de templates HTML)

H2 Database (banco de dados em memória para testes)

Bootstrap / HTML / CSS (interface do usuário)
úblico-Alvo

O sistema foi pensado para colaboradores da loja Clothes4Happiness, especialmente os responsáveis por estoque e administração de produtos.
Ele busca tornar as rotinas mais ágeis, seguras e transparentes, promovendo bem-estar tanto da equipe quanto dos clientes finais.

  Comunicação e Postura Profissional

Durante o desenvolvimento, buscamos adotar uma postura empática e colaborativa, levando em conta a visão do cliente.
Cada funcionalidade foi projetada pensando na usabilidade real da equipe e no impacto prático para o negócio.
